 EcoRide - Plateforme de Covoiturage Écologique

 Description:
Application web de covoiturage écologique développée dans le cadre du TP DWWM.

Prérequis:
- XAMPP (Apache + MySQL + PHP)
- Navigateur web moderne

Installation:
1. Installer XAMPP
2. Cloner le repository
3. Copier les fichiers dans `C:\xampp\htdocs\ecoride\`
4. Démarrer Apache et MySQL dans XAMPP
5. Importer la base de données (voir `database.sql`)
6. Visiter `http://localhost/ecoride/`
